# INFO634 Regional Crime Analysis Project

A data analysis project exploring regional crime patterns through summary statistics, visual analytics, time-series forecasting, and clustering.

This project was developed as a group project for INFO634 and demonstrates practical data analysis skills across exploratory analysis, visualisation, dimensionality reduction, and predictive modelling.

---

## Project overview

The project analyses regional crime-related data from multiple perspectives, including territorial authority patterns, offender-victim relationships, time trends, and regional clustering.

The goal is to turn raw regional crime records into interpretable insights that can support comparison across locations, crime types, victim groups, and time periods.

---

## Analysis areas

### Territorial authority analysis

- Summarises the percentage of unidentified offenders
- Visualises unidentified-offender percentages by region
- Builds cross-tabulations between crime location and crime type
- Uses heat maps to show relationships between locations and offence categories
- Compares victim age groups across different regions

### Relationship of offender and victim analysis

- Analyses the frequency distribution of offender-victim relationship categories
- Summarises serious crimes under different relationship groups
- Combines serious-crime counts with total cases to calculate severity ratios

### Time-series analysis

- Explores historical crime trends over time
- Builds a 12-month forecasting view to estimate future crime movement
- Helps identify trend direction and seasonality patterns where present

### Regional crime-pattern clustering

- Applies clustering to identify regions with similar crime-pattern profiles
- Uses PCA to reduce dimensionality for easier interpretation and visualisation
- Supports comparison of regional behaviour beyond simple count-based summaries
